## 🌐 What is CyberWorld?

**CyberWorld** is a fully original open-source cybersecurity-themed MMORPG and operations platform built by [FLLC Enterprise IT Solutions](https://www.fllc.net). Inspired by the depth of classic MMORPGs and the aesthetic of cyberpunk neon-galaxy worlds, CyberWorld transforms gameplay into a living SOC (Security Operations Center) experience.

> *"Enter the Grid. Hack the System. Defend the Network."*

### 🎮 Game Pillars
| Pillar | Description |
|---|---|
| **Cybersecurity MMORPG** | Skill trees around OSINT, penetration testing, network defense, and threat intelligence |
| **Midnight Neon Galaxy** | Immersive deep-space neon aesthetic — dark midnight backgrounds, electric cyan/magenta/purple highlights |
| **SOC Operations Center** | In-game Security Operations Center hub — monitor threats, respond to incidents, coordinate teams |
| **OSINT Intelligence** | Open-source intelligence gathering missions, social engineering scenarios, recon quests |
| **FLLC Theme** | Built on the FLLC Enterprise IT Solutions brand — professional, technical, futuristic |
| **Custom OS Feel** | Client UI designed to feel like a custom cybersecurity operating system terminal |

## 🛠️ Project Layout

- [`cache`](cache/src/main/java/net/runelite/cache) — Cache libraries for reading/writing game data
- [`runelite-api`](runelite-api/src/main/java/net/runelite/api) — CyberWorld API interfaces
- [`runelite-client`](runelite-client/src/main/java/net/runelite/client) — Game client with CyberWorld plugins & UI

## 🚀 Running the Client

```bash
# Open the project in IntelliJ IDEA as a Gradle project
# Run: net.runelite.client.RuneLite
./gradlew :runelite-client:run
```

## 🎨 CyberWorld UI Theme

The client uses the **Midnight Neon Galaxy** color palette:

| Token | Hex | Purpose |
|---|---|---|
| `BRAND_ORANGE` (Neon Cyan) | `#00DCFF` | Primary accent / brand |
| `DARKER_GRAY` | `#06060C` | Deepest background |
| `DARK_GRAY` | `#0A0A1C` | Panel background |
| `MEDIUM_GRAY` | `#1E2A4A` | Mid-tone UI |
| `PROGRESS_COMPLETE` | `#00FF88` | Success / neon green |
| `PROGRESS_ERROR` | `#FF1E50` | Error / hot pink |
| `PROGRESS_INPROGRESS` | `#9B00FF` | Electric purple |
